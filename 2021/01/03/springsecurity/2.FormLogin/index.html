FormLogin 인증과 UsernamePasswordAuthenticationFilter

작성일 2021-01-03 Edited on 2021-04-27 In SpringFramWork

FormLogin

Method 내용
loginPage(String loginPage) 사용자 정의 로그인 페이지
defaultSuccessUrl(String defaultSuccessUrl) 로그인 성공 후 이동 페이지
failureUrl(String authenticationFailureUrl) 로그인 실패 후 이동 페이지
usernameParameter(String usernameParameter) 아이디 파라미터명 설정
passwordParameter(String passwordParameter) 패스워드 파라미터명 설정
loginProcessingUrl(String loginProcessingUrl) 로그인 Form Action Url
successHandler(AuthenticationSuccessHandler successHandler) 로그인 성공 후 핸들러
failureHandler(AuthenticationFailureHandler authenticationFailureHandler) 로그인 실패 후 핸들러
